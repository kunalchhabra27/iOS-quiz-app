# Neon iOS Quiz App (Next.js)

A neon-styled quiz app that teaches iOS concepts with multiple-choice questions.

## Topics
- Swift Programming
- SwiftUI
- Combine Framework
- Xcode
- MVVM + Clean Architecture
- Swift Testing
- SwiftData

Each topic contains **10 MCQs** (70 total).

## Design
- Grid background
- Neon + red color palette
- Interactive answer feedback

## Run locally
```bash
npm install
npm run dev
```

Open `http://localhost:3000](https://ios-quiz-app.vercel.app`.
